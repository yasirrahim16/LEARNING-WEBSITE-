Commands / Concepts

Important notes

pwd

Current directory ka path dikhata hai.

ls

Folder ke andar files list karta hai.

ls -la

Detailed aur hidden files dikhata hai.

cd

Folder ke andar move hota hai.

mkdir

Naya folder banata hai.

touch

Nayi file create karta hai.

cp

File copy karta hai.

mv

File move ya rename karta hai.

rm

File delete karta hai.

cat

File content dikhata hai.

grep

Text search karta hai.

find

File search karta hai.

pkg update

Packages ki list refresh karta hai.

pkg upgrade

Installed packages update karta hai.

pkg install python

Python install karta hai.

termux-setup-storage

Storage permission set karta hai.

git clone

Repository download karta hai.

nano

Terminal editor kholta hai.

clear

Screen clear karta hai.

history

Recent commands dikhata hai.
